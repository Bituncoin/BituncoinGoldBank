# Bituncoin Gold Bank — BTNG Gold Coin

**Bituncoin Gold Bank** is Africa's sovereign digital banking platform powered by **BTNG Gold Coin (BTNGG)** — a gold-backed cryptocurrency serving Ghana and all 54 African nations.

Built with React Native and Expo for seamless cross-platform performance across iOS, Android, and Web.

## Platform Overview

**BTNG Gold Coin** is Africa's premier sovereign digital banking platform — a unified, gold-backed cryptocurrency and trading ecosystem serving all 54 African nations and the global diaspora.

🏅 **Gold-Backed Currency** — Every BTNG Gold Coin (BTNGG) is backed by physical gold. Real value, real sovereignty.

📈 **Complete Trading Platform** — Spot trading, P2P marketplace, copy trading, binary trading, and practice wallet.

🏦 **Sovereign Wallet** — BTNG Genesis Wallet with BIP-39 HD multi-account, MTN MoMo cash rail, QR code payments, and NFC tap-to-pay.

🌍 **Africa Free Trade Zone** — AfCFTA 54-nation trade engine with cross-border settlement in under 5 seconds.

🔐 **Security & Identity** — KYC verification, 2FA (TOTP), biometric lock, equity certificates, and certificate scanner.

## Getting Started

### 1. Install Dependencies

```bash
npm install
# or
yarn install
```

### 2. Start the Project

```bash
npm run start         # Start Expo development server
npm run android       # Launch Android emulator
npm run ios           # Launch iOS simulator
npm run web           # Start the web version
```

Reset the project (clear cache):

```bash
npm run reset-project
```

### 3. Lint the Code

```bash
npm run lint
```
